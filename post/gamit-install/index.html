解决方式

删掉高版本的gcc/gfortran

在/usr/bin下

1
2
3
4
5
6

sudo rm gcc
sudo rm gfortran
sudo apt-get install gfortran-5
sudo apt-get install gcc-5
sudo ln -s gfortran-5 gfortran
sudo ln -s gcc-5 gcc

gamit安装成功

ubnutu最新的也适用
